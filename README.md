# React app from scratch

It's repository for code from article 
[React app from scratch](https://medium.com/@evheniybystrov/react-app-from-scratch-d694300d1631)

## Docker

To build

    npm run docker:build
    
To run

    npm run docker:run
    
To stop

    docker:stop
  
## Run

    npm start
    
## Development

    npm run dev
    
## Testing

    npm test
